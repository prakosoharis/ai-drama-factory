# V2.1-B New Project Bootstrap

Create a clean native V2 creative project without copying engine code:

```sh
drama project create /path/to/ai-drama-pilot --project-id ai-drama-pilot --name "AI Drama Pilot" --template minimal-drama
```

The command refuses non-empty directories, requires `--force-empty-directory`
for an existing empty directory, and never overwrites a manifest. It creates
metadata, project documentation, scene-001, shot-001, production boundaries,
runtime directories, and `.drama-factory/bootstrap.json`. The bootstrap record
is small audit metadata suitable for Git; cache/runtime/locks/temp are ignored.

The project owns creative metadata and assets; the engine remains external.
Validate with `drama validate --project /path/to/ai-drama-pilot --level structure`.
No Git initialization, rendering, review mutation, candidate selection, or
assembly is performed.
